如何使用archinstall

發布於 2026-03-19 更新於 2026-03-23 分類於 linux ， ArchLinux 閱讀次數： 文章字數： 755 所需閱讀時間 ≈ 1 分鐘
用Archinstall安裝ArchLinux

archinstall是archlinux的安裝腳本。

在tty輸入archinstall後就可以開啟安裝的TUI畫面。

分類用途

分類 用途
Archinstall language Archinstall顯示的語言
Locales 鍵盤佈局、系統語言、編碼
Mirrors and respositories 鏡像站
Disk configuration 磁碟分區
Swap 交換空間
Bootloader 啟動載入程式
Kernels Linux核心
Hostname 裝置名稱
Authentication 登入驗證
Profile 裝置類型(Desktop\Server…)
Applications 藍牙、音效、影印服務
Network configuration 網路設定
Additional packages 附加應用程式
Timezone 時區
Automatic time sync (NTP) 自動時區同步

設定於都是使用鍵盤控制，方向鍵選擇上下，enter確定，esc返回。

設定

Archinstall language


在這裡可以設定archinstall顯示的語言，預設使用英文就好，不然會缺字

Locales

在這裡要設定鍵盤佈局、設定完之後的系統語言和編碼方式。維持預設不用修改(系統語言可以在安裝完成後再修改)。

Mirrors and respositories

這裡要選擇檔案下載的鏡像站，可以增加下載速度和穩定性。
先選擇Select regions。接著選擇Taiwan。按下/可以快速搜尋國家。

選擇完成後選back返回
