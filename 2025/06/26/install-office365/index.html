在windows上單獨安裝word、excel、powerpoint

發布於 2025-06-26 更新於 2026-03-23 分類於 Windows ， Office365 閱讀次數： 文章字數： 2.3k 所需閱讀時間 ≈ 2 分鐘
單獨安裝Office 365文書軟體

如果用Microsoft的Office 365官方安裝方式就只能夠一次安裝所有的Office軟體，無法單獨安裝Word、Excel、PowerPoint之類比較常用的軟體，但是真的會用到的通常就只有Word、Excel、PowerPoint，一次全部安裝反而很佔空間。所以如果要單獨安裝，就要使用Office Deployment Tool（ODT）。

下載Office Deployment Tool

前往ODT的網站下載Office 部署工具。
之後在隨意的資料夾執行下載的exe檔，這時會出現兩個檔案，分別是

  • configuration-Office365-x64.xml
  • setup.exe
    這兩個檔案，但是只需要setup.exe這個檔案，所以可以把configuration-Office365-x64.xml這個檔案刪除。

自訂安裝內容

前往Office 自訂工具這個網站。

  • 架構：通常選擇 64 位元。
  • 產品：通常是選擇 Microsoft 365 Apps 企業版
  • 更新通道：不用改
  • App 選擇：關閉不需要的軟體，開啟需要安裝的軟體，例如只需要安裝Word、Excel、PowerPoint的話就把這幾項開啟，其他的全部關閉
  • 語言：選繁體中文

之後就是一直點”下一頁”直到出現完成，
就可以按右上角的匯出按鈕。
檔案格式選Office Open XML
確認條款，並設定檔案名稱（例如 setting）
按匯出下載檔案

如果瀏覽器在下載xml檔時跳出了這類的警告

可以放心點擊保留或是確認。
